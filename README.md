# Monthly Budget Tracker

A command-line Python application that helps users manage their monthly finances. 

## What it does
- Collects monthly income and categorized expenses (rent, food, transportation, clothing, leisure)
- Calculates total spending, remaining balance, and savings rate
- Determines financial status: Saving, Overspending, or Breaking Even

## Built with
- Python

## How to run
1. Clone the repository
2. Run `python Project1-Monthly-Budget-Tracker.py` in your terminal
